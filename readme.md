LibraMeta is a privacy-focused, self-hosted search engine designed for efficient and detailed book metadata retrieval. It offers powerful filtering capabilities, intuitive search functionality, and prioritizes user privacy and security. 

Tech stack: Elasticsearch, SQL, Python, Flask, Jinja, HTML/CSS, Javascript

```
git clone git@github.com-sjf:sjf/librameta.git
```
